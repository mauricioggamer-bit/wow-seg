# 🐉 WoW Seg — Warband Tracker

Seguimiento semanal de personajes y tareas para World of Warcraft.

## Características

- 📋 Tracking de 44+ personajes organizados por warband
- ✅ Checklist de tareas semanales con cooldown
- 📥📤 Import/Export de datos en JSON
- ☁️ Sincronización con GitHub Gist
- 🔐 Acceso por contraseña (gusfraba + día)
- 🎨 UI temática de WoW Retail
- 🔄 Reset semanal automático (martes)

## Deploy

GitHub Pages — push a `main` despliega automáticamente.
